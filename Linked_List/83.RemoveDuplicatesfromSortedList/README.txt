******************************************************************************
Problem Statement: Remove Duplicates from Sorted List
******************************************************************************
Given a sorted linked list, delete all duplicates such that each element
appear only once.

For example,
Given 1->1->2, return 1->2.
Given 1->1->2->3->3, return 1->2->3.

******************************************************************************

******************************************************************************
Notes: 
******************************************************************************
Time Complexity : O(n)
Space Complexity : O(1)
