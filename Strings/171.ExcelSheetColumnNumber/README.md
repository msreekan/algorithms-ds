******************************************************************************
Problem Statement: [Excel Sheet Column Number](https://leetcode.com/problems/excel-sheet-column-number/)
******************************************************************************
Given a column title as appear in an Excel sheet, return its corresponding

column number.

For example:

    A -> 1

    B -> 2

    C -> 3

    ...

    Z -> 26

    AA -> 27

    AB -> 28 

*****************************************************************************

******************************************************************************
Notes: 
******************************************************************************
Time Complexity : O(n)

Space Complexity : O(1)

******************************************************************************
