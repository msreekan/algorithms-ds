******************************************************************************
Problem Statement: [Simplify Path](https://leetcode.com/problems/simplify-path/)
******************************************************************************
Given an absolute path for a file (Unix-style), simplify it.

For example,

path = "/home/", => "/home"

path = "/a/./b/../../c/", => "/c"


******************************************************************************

******************************************************************************
Notes: 
******************************************************************************
Time Complexity : O(n)

Space Complexity : O(1)

******************************************************************************
