******************************************************************************
Problem Statement: [Pascal's Triangle II](https://leetcode.com/problems/pascals-triangle-ii/)
******************************************************************************
Given an index k, return the kth row of the Pascal's triangle.

For example, given k = 3,

Return [1,3,3,1].
